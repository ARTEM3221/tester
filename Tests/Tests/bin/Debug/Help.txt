Справка по использованию программы "Экзаменатор".

МИНИМАЛЬНЫЕ СИСТЕМНЫЕ ТРЕБОВАНИЯ:
---------------------------------------------------------
- Операционная система Microsoft® Windows® XP SP3 (Windows Vista и Windows 7 официально не поддерживаются!)
- Программная платформа Microsoft .NET Framework 4.0
- Процессор Pentium III 1000 МГц или аналогичный Athlon XP
- 256 МБ оперативной памяти
- 1МБ свободного места на жестком диске.

ДЛЯ РАБОТЫ ПРОГРАММЫ
---------------------------------------------------------
Для работы программы на компьютере должна быть установлена свободно распространяемая платформа Microsoft .NET Framework 4.0
Скачать её можно на офф-сайте компании Microsoft. Также .NET Framework можно выбрать в списке устанавливаемого ПО при инсталляции программы.

ЕСЛИ ПРОГРАММА НЕ РАБОТАЕТ
---------------------------------------------------------
1) Проверить наличие на компьютере программной платформы Microsoft .NET Framework, если таковая отсутствует - установить.
2) Проверить, соответствуют-ли системные требования компьютера минимальным.
3) Проверить наличие папки "Tests" в корневом каталоге программы.
4) Проверить подпапки директории "Tests" на наличие и корректность имен.
